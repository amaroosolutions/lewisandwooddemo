Odoo 12 Sales order multi lots selection with available product quantity

Using this module you can multi lots in sales order with LOT available product quantity.

  • For using this module first enable Lots and Serial numbers from Inventory -> Configuration -> Lots Serial Numbers.


  • In the Product Checked Tracking By Lots.


  • As per screen create a Lot number and add quantity in the product inventory.


  • Here in screen you can see product with lot number and available product quantity..


  • As per screen create a Sale order and click on Lot number.


  • Once you click on Lot number you will see the Lot number with only avaiable Product quantity and you can select that Lot in the sale order.


  • Check View of lot with qty grater than 0.
  • Once Sale order will Confirm ,automatically get lot no from Sale Order and assign lot no and done quantity in delivery order. Click on Check Availability to Reserved Product qty.


  • Check here Initial Demand and Done Quantity of that Order.


  • Check here product Done Quantity in Picking.
